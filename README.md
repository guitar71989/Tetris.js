## Features

A browser Tetris game implemented with Javascript, jQuery, HTML5, and
CSS3.

## Controls:

  * &#2190; move block left
  * &#2192; move block right
  * &#2191; rotate block 90 degress clockwise
  * &#2193; soft drop
